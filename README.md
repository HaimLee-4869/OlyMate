# 🏟️ OlyMate (올리메이트)

> **공연의 감동을 완성하는 가장 스마트한 덕질 파트너** > 2025년도 국민체육진흥공단 공공데이터 활용 경진대회 서비스 개발 부문 출품작

## 💡 서비스 소개
OlyMate는 올림픽공원을 방문하는 K-POP 및 공연 관람객을 위해 **실시간 날씨, 공연 정보, 최적의 편의시설 동선, 주변 맛집 정보**를 통합 제공하는 AI 서비스입니다.

## 🚀 주요 기능
* **🤖 AI 시설 가이드:** 자연어(NLP)로 질문하면 화장실, 흡연장 등 편의시설 위치를 찾아줍니다. ("배고파", "물 마실 곳" 등)
* **🗺️ 스마트 맵:** 공연장별 맞춤 지도와 내 위치 기반 시설 찾기 기능을 제공합니다.
* **⛅ 실시간 정보:** 기상청 API와 연동된 정확한 날씨 및 공연 D-Day 카운트다운.
* **🍽️ 맛집 추천:** 공연 전후 혼잡도를 고려한 최적의 식사 장소 추천.

## 🛠️ 활용 데이터
1.  **기상청:** 단기예보 조회 서비스 API
2.  **한국체육산업개발주식회사:** 올림픽공원 공연 정보 API
3.  **서울올림픽기념국민체육진흥공단:** 올림픽공원 편의시설 정보, 서울올림픽파크텔 내외국인 객실 이용현황, 서울올림픽파크텔 식음료업장별 이용현황


## 💻 기술 스택
* **Language:** Python
* **Framework:** Streamlit
* **Map:** Folium
* **Data:** Pandas, Requests
